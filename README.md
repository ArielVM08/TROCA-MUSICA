# 🎵 Troca Música - Clone Spotify Style

Um player de música interativo inspirado na interface do Spotify, permitindo navegar entre faixas e controlar o fluxo das músicas de forma dinâmica.

## 🚀 Tecnologias Utilizadas
*   **HTML5**: Estruturação semântica.
*   **CSS3**: Estilização moderna com foco em UX.
*   **JavaScript**: Lógica de troca de faixas e controles de áudio.

## 🎧 Funcionalidades
- [x] Reprodução de músicas.
- [x] Troca de faixas (Próxima/Anterior).
- [x] Interface Responsiva.
- [x] Exibição de capas de álbuns/artistas.
